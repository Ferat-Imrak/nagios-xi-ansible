Ansible-Roles-Nagios
=========
This is a simple role which installs nagios-xi on virtual machine by using ansible playbook.
Requirements
------------
None
Role Variables
--------------
hostname="localhost.localdomain"
Dependencies
------------
None
Example Playbook
----------------
  roles:
    - role: ansible-role-apache
    - role: ansible-role-nagiosxi
